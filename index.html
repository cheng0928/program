生活科技教育前瞻計畫學習成果展覽

2018.12.19台大校園參訪

台灣大學創新設計學院為培養顛覆性創新、跨領域溝通的人才，讓跨科系的同 學共同合作，解決社會真實問題，在課程中特別加強：橫向溝通合作的應變能 力；資料量化分析；動手實作做精神；訊息傳達技巧；激發創意的領導力。 建立學生從觀察發現問題開始，到生活議題的探討，擴大國際視野。
Show project

2018.12.19 DSchool設計講座

「跨域實創教育廊帶」（D.school）營造生活化的學習情境，課程強調動手做，帶領學生觀察、發現問題，並培養解決問題的能力，把知識運用在生活上， 幫助孩子成為「生活設計師」。
Show project

2019.04.03北大高中校園參訪

北大高中為新北市生活科技第一代旗艦學校，由新北市教育局專案計畫補助， 建置「新奇教室(Thinking Classroom)」，讓學生思考如何客製屬於幼兒與長者 的生活輔具，「沒課空間(Maker Space)」來實作，透過各式機具加工，學生把 創意轉化成可行的用具，「智慧場域(intelligence site)」透過無線網路進行遠端 遙控，打造智慧家居的未來空間，並成為學生進行模擬與測試的重要場域。
Show project

2019.04.17螺鈿薪傳

螺鈿是很古老的傳統技藝，起源於商代的漆器，螺鈿工藝巔峰於清初，衰敗於 清末。明清這兩個漆器工藝高度發展的朝代國力鼎盛，到晚清，西方列強帶給 中國的文化衝擊，使得大家的焦點轉注在西方來的藝品技術與文化，包含螺鈿 工藝在內的許多傳統中國工藝技術，不再受到重視，逐漸沒落乃至消失。
Show project

2019.05.11校慶

「花開翔見、校慶美展」透過展出美術社全體成員的畫作，結合校慶與當日圖 書館改建落成，讓現場每位貴賓都能感受到他們所傳達的「生活舞感」。
