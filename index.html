生活科技教育前瞻計畫學習成果展覽

2018.12.19台大校園參訪

台灣大學創新設計學院為培養顛覆性創新、跨領域溝通的人才，讓跨科系的同 學共同合作，解決社會真實問題，在課程中特別加強：橫向溝通合作的應變能 力；資料量化分析；動手實作做精神；訊息傳達技巧；激發創意的領導力。 建立學生從觀察發現問題開始，到生活議題的探討，擴大國際視野。
Show project

2018.12.19 DSchool設計講座

「跨域實創教育廊帶」（D.school）營造生活化的學習情境，課程強調動手做，帶領學生觀察、發現問題，並培養解決問題的能力，把知識運用在生活上， 幫助孩子成為「生活設計師」。
Show project

2019.04.03北大高中校園參訪

北大高中為新北市生活科技第一代旗艦學校，由新北市教育局專案計畫補助， 建置「新奇教室(Thinking Classroom)」，讓學生思考如何客製屬於幼兒與長者 的生活輔具，「沒課空間(Maker Space)」來實作，透過各式機具加工，學生把 創意轉化成可行的用具，「智慧場域(intelligence site)」透過無線網路進行遠端 遙控，打造智慧家居的未來空間，並成為學生進行模擬與測試的重要場域。
Show project

2019.04.17螺鈿薪傳

螺鈿是很古老的傳統技藝，起源於商代的漆器，螺鈿工藝巔峰於清初，衰敗於 清末。明清這兩個漆器工藝高度發展的朝代國力鼎盛，到晚清，西方列強帶給 中國的文化衝擊，使得大家的焦點轉注在西方來的藝品技術與文化，包含螺鈿 工藝在內的許多傳統中國工藝技術，不再受到重視，逐漸沒落乃至消失。
Show project

2019.05.11校慶展演

「花開翔見、校慶美展」透過展出美術社全體成員的畫作，結合校慶與當日圖 書館改建落成，讓現場每位貴賓都能感受到他們所傳達的「生活舞感」。
Show project

2019.06.12Dschool教師專業社群讀書會

《專業之死》（The Death of Expertise）所指的，不是專業人士如今變得越來越 不專業，也不是在網路時代專業變得不再重要，而是指當前的社會越來越不尊 重──甚至鄙夷──專業知識與人士。
Show project

2019.06.17清大參訪

國立清華大學藝術學院成立「科技藝術研究中心」，主要目的為整合校內外跨 領域專業與資源，推動「數位藝術」、「數位文創」、「科技創新應用」三方 面的學術研究、教育推廣、產學合作與國際交流等，並致力於數位跨域轉繹創 作、數位文創促進以及科技創新激發。
Show project

2019.09.01生活舞感

板橋高中是新北市最優質的高中，以「多元識見、品創板圖」為學校願景，而近 年來，隨著平時生活步調的加快，升學與考試的壓力沉重，學生們容易對生活「 無感」，因此此次活動，希望可以讓人們對生活產生更多想法，打造屬於生活的 「舞感」。
Show project

2019.10.01設計思考融入校定必修

購買設計思考共用書籍，透過教師社群共備與增能，再帶領學生共讀書籍提出觀點並進行多元學習，以奠定教師推動設計思考教材使用模式，並厚實學生設計思考概念基礎。
Show project

2019.10.05開幕

本校於107學年爭取「實創Techshop生活科技教育前瞻領航旗艦級」計畫，籌畫「跨域實創教育廊帶(D.School)」，以符應108新課綱之「跨域素養課程模組」為基礎，進行空間整建與相關教學計畫推動。本校課程發展的重點是「生活舞感」主題，透過校內教師共備社群研習與合作，融入六大領域多種課程，以「設計思考」為共同核心課程，引導學生從多元課程中，重新觀察生活，發現問題，並且思考如何解決問題並且，靈活運用一般工具與教學空間的機具實踐構想，在學習過程中成就自我與珍愛環境。
Show project

2019.10.16社群會議

邀請本校社群教師討論實創教案與教學實踐相關問題，透過討論與成果報告、檢討反思等方式讓參加教師進行課程修正，以增進教師專業與推動實創課程。
Show project

了解四軸飛行器應用與飛行原理& 熟悉氣流對飛行器的姿態影響以及實際體驗無人機進行飛行與競賽。

2019.11.27金工

創辦金工教室，邀請對於金工有經驗的師傅，帶領校內老師實作飾品的設計、繪圖、製作，討論材料加工製作的方法與過程，透過實作讓參與課程之教師進行多元課程設計及學習，以增進教師專業與推動新課綱科技領域教學。
Show project

2019.12.14燃料電池模型車

在未來氫能經濟的社會，將利用再生能源產生氫氣，儲存至加氫站，供應燃料電池汽車交通工具及家用型發電機。藉此活動讓學生熟悉儲氫罐及燃料電池發電原理與應用，學習機械、化學觸媒及能源轉換效率等理化專業知識。
